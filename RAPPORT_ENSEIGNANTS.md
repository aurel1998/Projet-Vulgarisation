# Rapport : Site de Valorisation du BUT Science des Données
## Outil d'Orientation Interactive pour les Lycéens

---

## 1. Présentation et Contexte

### 1.1 Le Problème Identifié

L'orientation post-bac représente un défi majeur pour les lycéens de 15 à 18 ans. Face à la multiplication des formations et à la complexité des parcours, les jeunes manquent souvent d'outils concrets pour :
- **Comprendre** ce qu'est réellement une formation avant de s'y engager
- **Expérimenter** les compétences enseignées de manière ludique
- **Se projeter** dans les métiers et débouchés professionnels
- **Obtenir des réponses** à leurs questions sans barrière

Le BUT Science des Données (BUT SD) souffre particulièrement de cette opacité : perçu comme technique, mathématique ou réservé aux "experts", il effraie de nombreux lycéens qui pourraient pourtant y exceller.

### 1.2 La Solution Proposée

Ce site web interactif a été conçu pour **démystifier le BUT Science des Données** en permettant aux lycéens de :
- Découvrir la formation par la pratique (5 missions interactives)
- Comprendre les compétences acquises de manière concrète
- Visualiser les débouchés professionnels
- Obtenir des réponses personnalisées via un chatbot étudiant
- Se projeter dans un parcours adapté à leur profil

**Lien du site** : https://projet-vulgarisation.vercel.app

## 2. Fonctionnalités Principales

### 2.1 Page d'Accueil : Premier Contact Impactant

La landing page offre une **expérience immersive** dès l'arrivée :
- **Hero section** avec message clair : "La science des données, c'est comprendre le monde avec des chiffres"
- **Sections pédagogiques** :
  - "Pourquoi c'est fait pour toi" : démystification des idées reçues
  - Timeline des compétences : visualisation du parcours sur 3 ans
  - Mini-démo interactive : graphique de température (exemple concret)
  - Présentation des métiers : 8 débouchés avec salaires et outils
  - Témoignages étudiants : 4 retours d'expérience authentiques

**Objectif pédagogique** : Rassurer et donner envie en moins de 2 minutes.

### 2.2 Les 5 Missions Interactives : Apprendre en Faisant

Le cœur du site réside dans **5 missions gamifiées** qui permettent d'expérimenter les compétences du BUT SD :

#### Mission 1 : "Nettoie tes données"
- **Objectif** : Détecter et corriger les erreurs dans un dataset (doublons, valeurs manquantes, incohérences)
- **Compétence travaillée** : Nettoyage de données (data cleaning)
- **Format** : Tableau interactif avec feedback immédiat
- **Durée** : ~5 minutes

#### Mission 2 : "Qui va gagner le match ?"
- **Objectif** : Analyser des statistiques sportives et faire une prédiction
- **Compétence travaillée** : Analyse de données, calcul de scores, visualisation
- **Format** : Sliders interactifs + graphiques Recharts
- **Durée** : ~5 minutes

#### Mission 3 : "Netflix dans ta tête"
- **Objectif** : Créer un système de recommandation simple basé sur les goûts
- **Compétence travaillée** : Systèmes de recommandation, similarité, scoring
- **Format** : Sélection de tags + calcul de scores
- **Durée** : ~4 minutes

#### Mission 4 : "Parle en graphiques"
- **Objectif** : Choisir le bon type de graphique selon le contexte + détecter les graphiques trompeurs
- **Compétence travaillée** : Data visualisation, choix de représentation, esprit critique
- **Format** : Quiz interactif + graphiques Recharts (courbes, barres, camemberts)
- **Durée** : ~6 minutes

#### Mission 5 : "Détecte le spam"
- **Objectif** : Classifier des messages comme spam ou non-spam
- **Compétence travaillée** : Classification, analyse de texte, apprentissage automatique
- **Format** : Interface de classification avec feedback
- **Durée** : ~5 minutes

**Points forts pédagogiques** :
- ✅ **Feedback immédiat** : L'élève comprend instantanément ses erreurs
- ✅ **Gamification** : Badges, progression, scores de qualité
- ✅ **Concrétisation** : Chaque mission illustre une compétence réelle du BUT SD
- ✅ **Accessibilité** : Aucun prérequis technique nécessaire

### 2.3 Chatbot Étudiant : Réponses Personnalisées

Un **chatbot rule-based** simule un étudiant en BUT SD qui répond aux questions courantes :
- Questions sur les prérequis ("Faut-il savoir coder ?")
- Questions sur le programme ("Qu'est-ce qu'on apprend ?")
- Questions sur les débouchés ("Quels métiers après ?")
- Questions sur l'admission ("Comment postuler ?")

**Avantages** :
- Disponible 24/7, sans barrière
- Réponses adaptées au langage des lycéens
- Liens vers les pages pertinentes du site

### 2.4 Simulateur de Parcours : Orientation Personnalisée

Un **outil d'orientation interactif** en 3 questions :
1. Type de bac (Général / Technologique)
2. Centre d'intérêt (Sport / Réseaux / Environnement / Marketing)
3. Préférence d'apprentissage (Projets / Théorie / Mix)

**Résultat** : Recommandation personnalisée avec :
- 3 missions à faire en priorité
- 2 métiers correspondant au profil
- 3 conseils concrets pour réussir

**Valeur pédagogique** : Aide à la décision d'orientation basée sur le profil de l'élève.

### 2.5 Pages Informatives Complémentaires

- **Débouchés** : 8 métiers détaillés (salaire, outils, perspectives)
- **La Formation** : Programme vulgarisé, timeline, stages, alternance
- **FAQ** : 15 questions fréquentes en accordéon
- **Shorts** : Vidéos courtes (format TikTok) sur le quotidien en BUT SD
- **Candidater** : Guide de candidature et formulaire de contact

## 3. Utilisation Pédagogique

### 3.1 Scénarios d'Utilisation en Classe

#### Scénario 1 : Découverte en Autonomie (1h)
**Objectif** : Permettre aux élèves de découvrir le BUT SD de manière autonome

**Déroulement** :
1. Présentation du site (5 min) : L'enseignant présente l'outil et ses objectifs
2. Exploration libre (30 min) : Les élèves naviguent sur le site, font 1-2 missions
3. Partage d'expérience (15 min) : Discussion sur ce qu'ils ont découvert
4. Retour écrit (10 min) : Chaque élève note 3 choses apprises

**Bénéfices** :
- Découverte active et personnalisée
- Pas de pression, chacun avance à son rythme
- Feedback immédiat des missions

#### Scénario 2 : Atelier Orientation (2h)
**Objectif** : Utiliser le site comme support pour un atelier d'orientation

**Déroulement** :
1. **Phase 1 - Découverte** (30 min) : Navigation libre sur le site
2. **Phase 2 - Simulateur** (20 min) : Chaque élève utilise le simulateur de parcours
3. **Phase 3 - Missions** (40 min) : Réalisation de 2-3 missions en petits groupes
4. **Phase 4 - Débat** (30 min) : Discussion sur les compétences découvertes et les métiers

**Bénéfices** :
- Approche ludique qui dédramatise l'orientation
- Expérience concrète des compétences
- Création d'un dialogue entre pairs

#### Scénario 3 : Support pour Entretiens d'Orientation
**Objectif** : Utiliser le site comme support lors d'entretiens individuels

**Déroulement** :
- L'enseignant guide l'élève vers les sections pertinentes selon ses questions
- Utilisation du chatbot pour répondre aux questions courantes
- Réalisation d'une mission ensemble pour illustrer une compétence

**Bénéfices** :
- Support visuel et interactif
- Démystification par la pratique
- Création d'un langage commun

### 3.2 Utilisation en Accompagnement Personnalisé

Le site peut être utilisé dans le cadre de l'**Accompagnement Personnalisé (AP)** ou des **heures de vie de classe** :

- **Séance "Découverte des métiers du numérique"** : Focus sur la page Débouchés
- **Séance "Comprendre les formations"** : Focus sur la page Formation + Simulateur
- **Séance "Esprit critique et données"** : Focus sur la Mission 4 (graphiques trompeurs)

### 3.3 Utilisation à Distance

Le site est **accessible 24/7** et peut être utilisé :
- En devoir à la maison : "Explore le site et fais au moins 2 missions"
- En préparation d'un forum des métiers : Découverte avant l'événement
- En complément d'une visite d'établissement : Consolidation après la visite

## 4. Impact et Bénéfices pour les Lycéens

### 4.1 Démystification de la Formation

**Avant** : "Le BUT SD, c'est pour les matheux, faut savoir coder, c'est trop technique"

**Après** : "Ah, c'est comprendre des données, nettoyer des tableaux, faire des graphiques. Je peux essayer !"

Le site **brise les barrières psychologiques** en montrant que :
- Aucun prérequis n'est nécessaire
- Les compétences sont accessibles et concrètes
- On apprend progressivement, pas besoin d'être expert dès le départ

### 4.2 Expérience Concrète des Compétences

Contrairement à une brochure ou une présentation PowerPoint, le site permet de **manipuler réellement** les outils et concepts :
- Nettoyer des données (Mission 1)
- Faire des prédictions (Mission 2)
- Créer un système de recommandation (Mission 3)
- Choisir le bon graphique (Mission 4)
- Classifier des données (Mission 5)

Cette **expérience pratique** permet aux lycéens de :
- Se projeter dans la formation
- Évaluer leur intérêt réel
- Comprendre ce qu'ils vont apprendre

### 4.3 Aide à la Décision d'Orientation

Le **simulateur de parcours** offre une aide concrète à la décision :
- Recommandations personnalisées selon le profil
- Visualisation des métiers correspondants
- Conseils adaptés au type de bac et aux préférences

**Impact** : Réduction de l'anxiété liée au choix d'orientation.

### 4.4 Accessibilité et Inclusivité

Le site est conçu pour être **accessible à tous** :
- Interface intuitive, pas besoin de compétences techniques
- Langage simple et vulgarisé
- Exemples concrets (sport, Netflix, météo)
- Disponible sur mobile, tablette et ordinateur

**Résultat** : Aucun élève n'est exclu par manque de prérequis.

### 4.5 Création d'un Dialogue

Le site **facilite le dialogue** entre :
- Les lycéens et leurs enseignants (support visuel pour les discussions)
- Les lycéens et leurs parents (outil de compréhension partagée)
- Les lycéens entre eux (expérience commune à partager)
